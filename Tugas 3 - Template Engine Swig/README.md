## Cara Menjalankan:
1. Clone repository ini, buka folder dan masuk terminal
2. Jalankan `npm install express --save`
3. Jalankan `npm install`
4. Jalankan `node app.js`
5. Buka browser menuju ke halaman http://localhost:3000
